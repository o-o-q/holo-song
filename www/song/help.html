help

  • list
    • ホロライブ 全歌動画(youtube) の再生数を週間ランキングで掲載しています
    • 毎週木曜 23:45 に更新されます
    • 再生数が緑のものは 新着の動画 です
      • 集計期間は 1週未満となります
  • filter
    • 動画タイトルの文字列に対してマッチングを行い, リストを絞り込むことができます
    • and 条件
      • 半角スペース
    • or 条件
      • , ( カンマ )
      • , の両端にはスペースを挟まない
      • , を絞り込み文字列とすることはできない
    • 空白(and) と , (or) が混在する場合は , (or) 条件の結合が優先されます
      • 例えば, すいせい と azki の オリジナルソング を絞り込みする場合は
        すいせい,azki original,オリジナル
        のように入力します
    • それ以外の複雑な入れ子条件はできません
    • オリジナル曲のみにする場合
        • original,オリジナル
        と入力すれば, おおむね絞り込まれます
    • カバー曲のみにする場合
        • cover,カバー
        と入力すれば, おおむね絞り込まれます
  • player
    • リストのタイトルをクリックで, その曲から5曲を再生
    • 5曲の再生が終わったら, ランダムに 別の 5曲を再生
  • source
    • github
    • 抽出対象の youtube リストID
